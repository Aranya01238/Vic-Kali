# SECOND BRAIN - Your External Memory System

Diya is now your comprehensive second brain for capturing, organizing, and retrieving knowledge.

QUICK START COMMANDS:

1. NOTE-TAKING
   - "note: I just learned about async/await" - Add a note (auto-tagged)
   - "search notes for python" - Find notes by keyword
   - "my notes on programming" - Get all notes on a topic

2. DAILY INSIGHTS & LEARNING
   - "insight: Context-driven decisions work better" - Record a discovery
   - "today insights" - See today's insights
   - "week summary" - Get weekly insights summary
   - "learned that sleep improves memory" - Log something you learned
   - "learning log" - View what you've learned recently

3. HABIT TRACKING
   - "add habit: 30 minutes meditation" - Start tracking a habit
   - "my habits" - Check your habit status and streaks
   - "completed meditation" - Mark a habit as done today

4. PROJECT MANAGEMENT
   - "start project: Complete the AI system" - Create a project
   - "my projects" - View active projects with progress
   - "update progress to 50%" - Update project progress

5. BRAIN STATUS
   - "brain status" - See total notes, habits, projects, insights
   - "second brain" - Get comprehensive brain statistics

FEATURES:

✓ Automatic Tag Extraction

- Uses #hashtags in notes for quick organization

✓ Topic-Based Organization

- Notes grouped by topic (programming, health, learning, etc.)
- Quick recall by topic

✓ Insight Generation

- Daily insights tracked separately
- Weekly summaries to identify patterns
- Category-based organization

✓ Habit Streaks

- Tracks daily consistency
- Visual progress in "my habits" view
- Frequency support: daily, weekly, monthly

✓ Project Progress Tracking

- Visual progress bars
- Automatic status: active, paused, completed
- Description and tagging

✓ Learning Log

- Record what you learned and where
- Topic-based filtering
- Source attribution

TIPS FOR EFFECTIVE SECOND BRAIN USAGE:

1. Use consistent topics for notes (e.g., "programming", "health", "ideas")
2. Add #hashtags naturally in notes for better discovery
3. Review "week summary" weekly to consolidate learning
4. Track habits you actually want to build (start with 1-2)
5. Update project progress regularly for momentum
6. Use "learned that" to externalize knowledge
7. Search notes when you can't remember something
